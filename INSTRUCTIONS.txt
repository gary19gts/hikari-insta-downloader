================================================================================
                            SYSTEM REQUIREMENTS
================================================================================

- Operating System: Windows 10+, macOS 10.14+, or Linux (Ubuntu 18.04+)
- Python: Version 3.7 or higher
- RAM: 512MB minimum, 1GB recommended
- Storage: 100MB for application + space for downloads
- Internet: Stable connection required

================================================================================
                              INSTALLATION
================================================================================

METHOD 1: AUTOMATIC INSTALLATION (Recommended)
----------------------------------------------
1. Open Command Prompt/Terminal in the application folder
2. Run: python install.py
3. Wait for all dependencies to install
4. Done! Skip to "RUNNING THE APPLICATION" section

METHOD 2: MANUAL INSTALLATION
-----------------------------
1. Install Python dependencies:
   python -m pip install -r requirements.txt

2. Install download engines:
   python -m pip install yt-dlp instaloader gallery-dl

3. Verify installation:
   python test_installation.py

METHOD 3: SMART LAUNCHER (Auto-checks and installs)
---------------------------------------------------
1. Run: python launch.py
2. Follow on-screen prompts to install missing components
3. Application will launch automatically when ready

================================================================================
                              HOW TO USE
================================================================================

STEP 1: GET INSTAGRAM URL
-------------------------
- Go to Instagram in your web browser
- Navigate to the post, reel, or story you want to download
- Copy the URL from the address bar
- Supported formats:
  * Posts: https://www.instagram.com/p/[POST_ID]/
  * Reels: https://www.instagram.com/reel/[REEL_ID]/
  * Stories: https://www.instagram.com/stories/[USER]/[STORY_ID]/

STEP 2: PASTE URL IN APPLICATION
--------------------------------
- Launch Hikari Insta Downloader
- Paste the Instagram URL in the "Instagram URL" field
- The application will automatically detect the content type

STEP 3: SELECT DOWNLOAD ENGINE
------------------------------
Choose the best engine for your content:

üì∏ INSTALOADER (Default - Best for Photos)
- Excellent for image posts and photo carousels
- Preserves metadata and high quality
- Instagram-specialized features

üé• YT-DLP (Best for Videos)
- Superior for video content and reels
- Best video quality options
- Most reliable for video downloads

üåê GALLERY-DL (Multi-platform)
- Good for general use
- Supports multiple social platforms
- Consistent performance

STEP 4: CHOOSE OUTPUT FOLDER
----------------------------
- Default: Downloads/ folder in application directory
- Click "Browse" to select a custom folder
- Click "Open" to view the current output folder
- Ensure you have write permissions to the selected folder

STEP 5: DOWNLOAD CONTENT
------------------------
- Click "Download Content" button
- Monitor progress in the progress bar
- Wait for "Download completed successfully!" message
- Files will be saved to your chosen output folder

WHEN TO USE EACH ENGINE:

================================================================================
                              TROUBLESHOOTING
================================================================================

PROBLEM: "Engine not found" error
SOLUTION: 
- Run: python install.py
- Or manually: pip install yt-dlp instaloader gallery-dl

PROBLEM: Download fails
SOLUTIONS:
- Verify the Instagram URL is correct and accessible
- Check your internet connection
- Try a different download engine
- Run diagnostics: Click "Run Diagnostics" button
- Ensure the content is publicly accessible

PROBLEM: Permission errors
SOLUTIONS:
- Choose a different output folder (Desktop, Documents)
- Run as administrator (Windows) or with sudo (Linux/macOS)
- Check folder write permissions

PROBLEM: Slow downloads
SOLUTIONS:
- Check your internet connection speed
- Try a different download engine
- Close other applications using internet
- Try downloading during off-peak hours

PROBLEM: Application won't start
SOLUTIONS:
- Verify Python 3.7+ is installed: python --version
- Install dependencies: python install.py
- Check for error messages in terminal
- Run diagnostics: python test_installation.py
